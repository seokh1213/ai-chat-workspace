# Backend Template

새 프로젝트의 Kotlin Spring Boot backend를 생성할 때 이 템플릿을 기준으로 시작한다.

초기 구현 목표:

- workspace CRUD
- data space CRUD
- source record CRUD
- chat session/message API
- SSE event stream
- AI provider abstraction
- operation apply/checkpoint

다음 AI Agent는 `../../agents/IMPLEMENTATION_PLAN.md`를 따른다.
